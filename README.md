# 🌐 Dolla Grace Ambwaya — Portfolio

> "Lifestyle, but make it worth it."

A modern, futuristic personal portfolio built with Next.js 16, TypeScript, and Tailwind CSS. Features a dark theme with violet gradients and teal accents.

**Live Site:** [dollagraceambwaya-commits-github-io.vercel.app](https://dollagraceambwaya-commits-github-io.vercel.app)

---

## ✨ Sections

- **Navbar** — Fixed navigation with mobile menu
- **Hero** — Animated gradient intro with CTA buttons
- **About** — Background, tagline, and quick info card
- **Skills** — Frontend, Backend, Tools, and Marketing skills
- **Projects** — Real project contributions with GitHub links
- **Contact** — GitHub, LinkedIn, Instagram, and Email links

---

## 🛠 Tech Stack

- **Framework:** Next.js 16 (Turbopack)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Deployment:** Vercel
- **Version Control:** Git + GitHub

## 🚀 Run Locally

```bash
git clone https://github.com/dollagraceambwaya-commits/dollagraceambwaya-commits.github.io.git
cd dollagraceambwaya-commits.github.io
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
